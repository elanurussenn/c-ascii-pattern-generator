# C ASCII Pattern Generator

Bu proje, C programlama dili kullanılarak geliştirilmiş bir konsol uygulamasıdır.
Kullanıcıdan alınan boyut değerine göre yıldız karakterleriyle ASCII desen çıktıları oluşturur.

##  Projenin Amacı

Bu çalışma; for döngüleri, if/else koşul yapıları, kullanıcıdan veri alma ve konsol çıktısı üretme gibi temel C programlama konularını pekiştirmek amacıyla hazırlanmıştır.

##  Kullanılan Teknolojiler

* C
* GCC Compiler
* Code::Blocks IDE

##  Özellikler

* Kullanıcıdan boyut değeri alma
* Yıldız karakterleriyle desen oluşturma
* Döngü ve koşul yapılarıyla şekil üretme
* Konsol ekranında ASCII tabanlı çıktı gösterme

##  Dosyalar

* `main.c` - Projenin ana kaynak kod dosyası

##  Çalıştırma

```bash
gcc main.c -o pattern
./pattern
```

Windows için:

```bash
gcc main.c -o pattern.exe
pattern.exe
```
